# 🌿 Беларусь: Знай свой край
### Многопользовательская викторина-баттл на C# · WPF · SignalR

---

## Требования

- **Windows 10/11** (64-bit)
- **.NET 8 SDK** → https://dotnet.microsoft.com/download/dotnet/8.0
- **Visual Studio 2022** (или Rider) — опционально

---

## Быстрый старт

### 1. Открыть в Visual Studio
Открыть файл `BelarusQuiz.sln` → Visual Studio сам восстановит пакеты.

### 2. Запустить через скрипты (без IDE)

**Вариант А — через .bat файлы:**

Откройте две консоли (cmd или PowerShell):

```
# Консоль 1 — Сервер
start_server.bat

# Консоль 2 — Клиент
start_client.bat
```

**Вариант Б — вручную:**
```bash
# Сервер
cd BelarusQuiz.Server
dotnet run

# Клиент (в другом окне)
cd BelarusQuiz.Client
dotnet run
```

## Как играть

### Режим Соло (без сервера)
1. Запустите клиент: `start_client.bat`
2. Введите никнейм
3. Нажмите **«Игра против бота»**
4. Отвечайте на 10 вопросов о Беларуси быстрее бота!

### Мультиплеер (нужен сервер)
1. Запустите сервер: `start_server.bat`
2. Запустите клиент(ы): `start_client.bat`
3. **Игрок 1:** нажать «Быстрая игра» → получит код комнаты
4. **Игрок 2:** нажать «Подключиться» → ввести код → войти
5. Оба нажимают **«Я готов»** → игра начинается автоматически

### Сетевая игра (разные компьютеры)
1. Запустить сервер на одном компьютере
2. Узнать IP этого компьютера (команда `ipconfig`)
3. На других компьютерах в поле «Адрес сервера» ввести: `http://192.168.X.X:5000`

## Структура проекта

```
BelarusQuiz/
├── BelarusQuiz.Shared/    # Общие модели (используются сервером и клиентом)
├── BelarusQuiz.Server/    # Сервер (ASP.NET Core + SignalR)
│   ├── Hubs/GameHub.cs    # Ядро мультиплеера
│   ├── Services/          # Логика комнат, счёта
│   └── Data/QuestionBank  # 20 вопросов о Беларуси
└── BelarusQuiz.Client/    # Клиент (WPF)
    ├── Views/             # Все экраны (XAML)
    ├── Services/          # SignalR подключение, навигация
    └── Resources/         # Тема, цвета, стили
```

## Вопросы в игре (темы)
- 🗺 **География** — реки, озёра, города, области
- 📜 **История** — даты, события, личности
- 🌿 **Природа** — парки, животные, экология
- 🎨 **Культура** — искусство, традиции, кухня, архитектура
- 🏛 **Символика** — герб, флаг, гимн

## Система очков

| Действие | Очки |
|---|---|
| Правильный ответ | +100 |
| Бонус за скорость | +0..+50 (зависит от времени) |
| Бонус серии (streak ×2) | +10 за каждый в серии, макс +50 |
| Неверный ответ | 0, серия обнуляется |
